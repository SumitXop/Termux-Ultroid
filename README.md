# Termux-ubuntu 

# install Ultroid

```
cd Termux-Ultroid
bash ubuntu-start.sh

Now Open ubuntu
apt update && apt upgrade
apt install --fix-broken
apt install git
git clone https://github.com/rooted-cyber/Termux-Ultroid
cd Termux-Ultroid
bash install-ultroid.sh
bash start-ultroid.sh
```
